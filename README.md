# Welcome To **AdrielOS**!

## Here's How to Get Started [Works for school iPads]!:
- [Install and use AdrielOS](##getting-started)





## Getting Started
1. Install Swift Playgrounds in Self Service or App Store.
2. Download AdrielOS
3. Open AdrielOS in Swift Playgrounds
4. Select "Open Anyways", *if prompted*
5. Tap the **play button** at the top of the code interface.
6. Wait for it to load *(this may take a few minutes)*
7. Done!
